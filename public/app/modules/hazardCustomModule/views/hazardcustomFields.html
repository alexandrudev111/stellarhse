Instructions

  • 1. Drag and drop custom fields into the highlighted area.
  • 2. Select any "change me" text to edit it.
  • 3. Use the Add buttons to add to list items.
  • 4. Add titles and text for help pop-ups on the right. Leave them as is if you don't want to add pop-ups.
  • 5. Change the order of the fields by changing the order numbering.
  • 6. Delete fields by clicking on thebeside the field.
  • 7. Save and preview your changes.
  • 8. To change the heading "Custom Fields", go to Manage Form Fields, search for that field heading in the table, open and edit it. Just make sure you pick the one for the tab you are customizing
